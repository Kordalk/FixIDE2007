A plugin for Delphi 2007 (BDS 5.0) that returns a detailed, clickable compilation error log in Windows 10. 
LogView line highlighting is implemented based on the compiler status. The plugin is installed as a .DPK file, 
integrated into the environment in place of the standard Messages window, and added to the "Tools" system menu. 
This allows you to enable or disable it. To jump to a specific line in the code, press CTRL + MB Left.
